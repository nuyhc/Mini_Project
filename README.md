### 8. 주요 질병 사망자 및 보건 환경 비교 분석 (CAoMDDaHE)

4. 의약품 판매 소비 - 손유선
5. 공공사회 복지 지출 - 박이슬
6. 종합 및 결론 - 김채현, 박이슬
